Advanced Operating Systems

Course project developed as part of the Advanced Operating Systems course

Overview

This project was completed as part of the Advanced Operating Systems course offered by the Systems Group at ETH Zürich. The objective was to gain practical experience in operating system design by implementing core services on top of the Barrelfish capability-based microkernel.

The project covered virtual memory, page fault handling, process management, message passing, multicore support, UMP communication, and name service functionality.

Course Context

  • Course: Advanced Operating Systems
  • Institution: ETH Zürich
  • Year: 2025
  • Offered by: Systems Group

Technologies

  • C Programming Language
  • Barrelfish capability-based microkernel
  • Multicore systems and inter-process communication

Learning Outcomes

The project provided hands-on experience with low-level systems programming, including virtual memory, process isolation, concurrency, and communication between distributed system components. It also involved extending and debugging a complex operating system codebase.

Results

Implemented core operating system services in Barrelfish, including memory management, process handling, fault handling, and communication mechanisms. The project strengthened practical understanding of modern OS architectures and multicore system design.
